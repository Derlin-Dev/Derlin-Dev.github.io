Proyecto API de Tareas

👋 Sobre el proyecto

Esta es una API para gestionar tareas. Permite crear, listar, actualizar y eliminar tareas de los usuarios registrados. Cada usuario tiene sus propias tareas y imprementa autenticación con JWT y Spring security.

Tecnologías Utilizadas

  • ☕️ Java 21
  • 🌱 Spring Boot
  • 🔐 Spring Security con JWT
  • 🐘 PostgreSQL
  • 🐳 Docker y Docker Compose
  • 📦 JPA/Hibernate

Funcionalidades

  • Registro de usuarios
  • Autenticación con JWT
  • CRUD de tareas (Crear, Leer, Actualizar, Eliminar)
  • Asociación de tareas a usuarios específicos

Requisitos

  • Docker
  • Docker Compose

Instalación y Ejecución

  1. Clona el repositorio:
    git clone https://github.com/usuario/proyecto-tareas.git
    cd proyecto-tareas
  2. Construye y levanta los contenedores con Docker Compose:
    docker-compose up --build
  3. Accede a la API en http://localhost:8080.

Variables de Entorno

El archivo docker-compose.yml está configurado para tomar las siguientes variables de entorno:

  • JWT_SECRET: Clave secreta para firmar los tokens JWT.
  • DATABASE_URL: URL de la base de datos PostgreSQL.
